# Установка
Скачиваем ffmpeg-git-full.7z
https://www.gyan.dev/ffmpeg/builds/
И распаковываем. Меняем название папки с самими файлами на ffmpeg и кидаем в диск C.

Теперь открываем
панель управления – Система – Свойства системы – переменные среды – Path – изменить – создаём новую переменную к папке bin скачанного архива к примеру здесь это C:\ffmpeg\bin - Ок Ок Ок Закрыть

На данном этапе уже будет работать exe файл https://t.me/kiberdans/27 (сюда не загрузил из за размера), но если вам надо python то установите библиотеки. ↓

Качаем файлы из этого гитхаба, и в строке проводника пишем cmd. В открывшейся командной строке пишем:
```shell
pip install -r requirements.txt
```

Теперь всё должно работать. Так же обязателен https://github.com/Flowseal/zapret-discord-youtube .
